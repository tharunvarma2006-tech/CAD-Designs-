# Wheel Motor Housing

## Overview
Designed a modular wheel motor housing for rover applications.

## My Contributions
- Designed internal supports and mounting structures.
- Integrated ventilation features.
- Fabricated and tested multiple prototypes.

## Results
- Improved durability and maintainability.
- Enabled rapid motor replacement.
