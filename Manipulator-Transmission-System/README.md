# Manipulator Transmission System

## Overview
Designed transmission systems for shoulder, base rotation, and end-effector actuation of a 5-DoF robotic arm.

## My Contributions
- Designed 10:1 and 20:1 worm-drive mechanisms.
- Developed a three-way bevel gear mechanism with a 2:1 herringbone gear stage.
- Optimized structures for reduced weight and improved performance.

## Key Specifications
- Base Worm Drive: 10:1
- Shoulder Worm Drive: 20:1
- End-Effector Transmission: Three-Way Bevel Gear System
- Payload Capacity: 8 kg

## Results
- Reduced arm weight by 1.5 kg.
- Maintained minimum safety factor above 2.5.
- Enabled pitch and roll motion through a single mechanism.
